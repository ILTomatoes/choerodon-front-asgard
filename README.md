# choerodon-front-asgard

Choerodon Front Asgard 是一个任务调度服务(通过`saga` 实现微服务之间的数据一致性)前端。

## Start

``` bash
cd choerodon-front-asgard
npm install
chmod -R u+x node_modules
npm start
```
